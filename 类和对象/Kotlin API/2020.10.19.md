- `object`对象声明：使用关键字`object`来进行对象声明，对象声明出来的对象都是单例的。`object`关键字后面跟随者对象声明的名称，对象声明不是一个表达式,不能用在赋值语句的右边。对象声明的初始化过程是**线程安全的并且在首次访问时**进行。如果需要引用该对象声明的话,可以直接使用其名称即可:
  
      ```kotlin
            object His2InpationMapper {
                fun transform() {

                }
            }
      ```
